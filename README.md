# SimpleCompiler
## 概述

SimpleCompiler是为SysY语言构建的编译器，其基于Rust编写，包括前端、中端、和后端，

2024.6.7 决心抛弃掉ZLJ@NWPU的框架，仿照北大文档，另起炉灶完成一个编译器。

- [ ] IR生成与符号表
